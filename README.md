# 模运算工具箱 (ModApp)

一个基于 Python + Flet 开发的数论与密码学小工具。

## 功能

- 模运算
- 幂模运算
- 素数判断
- 欧拉函数
- 最大公因数
- 原根相关计算
- 数论学习模块

## 技术栈

- Python
- Flet
- Flutter(Web)

## 本地运行

安装依赖：

```bash
pip install -r requirements.txt
```

运行：

```bash
flet run --web
```
